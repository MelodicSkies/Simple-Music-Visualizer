--------------------------------------------------------------- Simple Music Visualizer ---------------------------------------------------------------

Simple visualizer for real time audio input
Uses FFT to plot amplitude vs frequency data

-------------------------------------------------------------------------------------------------------------------------------------------------------

Release V 1.0:
--------------
> classic eq style visualizer for music
> implemented exponential smoothing (optional)
> eliminated y and x axis to generate a more streamlined look (can be turned on if needed)
> test.py contains code for testing different fft calculations, based on matplotlib instead of pyqtgraph
